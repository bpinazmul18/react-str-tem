### Start dev

1. Install pnpm https://pnpm.io/installation
   ```bash
   npm install -g pnpm
   ```
2. Install dep
   ```bash
   pnpm i
   ```
3. Start dev server
   ```bash
   pnpm start
   ```
   > Please use domain localhost instead of ip address

### Deploy

1. Build dist, then you should be able to find dist files in ./dist/
   ```bash
   pnpm build
   ```
